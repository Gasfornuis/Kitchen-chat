## 🛠️ Installation & Setup

### 🚀 Quick Start
```bash
# Clone the repository
git clone https://github.com/Gasfornuis/Kitchen-chat.git
cd Kitchen-chat

# Install dependencies (if using local development)
pip install -r requirements.txt

# Setup environment variables
echo 'FIREBASE_SERVICE_ACCOUNT="{...}"' > .env
echo 'FIREBASE_SECRET="your-secret"' >> .env

# Start local development
vercel dev
```

### 🌐 Environment Variables
```env
FIREBASE_SERVICE_ACCOUNT='{"type":"service_account",...}'
FIREBASE_SECRET='your-firebase-secret-key'
```

### 📱 PWA Installation
1. Visit [kitchen-chat.vercel.app](https://kitchen-chat.vercel.app)
2. Look for "Install App" prompt
3. Enjoy native app experience!
